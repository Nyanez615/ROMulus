# Privacy Policy

**Last updated:** 2026-05-30

## Summary

ROMulus is a local file management tool. In its current version (V1), **no data leaves your device under any circumstances.**

## What ROMulus does NOT do

- Does not collect, upload, or transmit ROM files or their contents
- Does not share data with third parties
- Does not include advertising or tracking
- Does not identify you personally

## Data stored locally

ROMulus stores the following data **only on your device**, in your system's application data directory:

- Your language and region preferences
- ROM folder paths you have configured
- Action history (files moved, deleted, or kept)
- Cached game metadata fetched from public APIs (IGDB, SteamGridDB)
- Cached cover art images

## Third-party API requests (optional features)

When you use metadata enrichment or cover art features, ROMulus makes requests to:

- **IGDB** (igdb.com) — game titles and metadata. See [IGDB Privacy Policy](https://www.igdb.com/privacy_policy).
- **SteamGridDB** (steamgriddb.com) — cover art. See [SteamGridDB Privacy Policy](https://www.steamgriddb.com/privacy-policy).

Only game titles and console names are sent. No file paths or personal information are included in these requests.

## Crash reporting (opt-in only)

If you choose to enable crash reporting during onboarding, anonymous crash data is sent to Sentry. This includes stack traces only — no file paths, ROM titles, or personal data. You can disable this at any time in Settings.

## V2 cloud sync (future)

A future version may offer optional cloud sync of preferences and metadata. This will be strictly opt-in, and ROM file content will never be synced. A full update to this policy will be published before that feature is available.
